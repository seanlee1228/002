## 第一章 制度总则

### 1.1 核心目标

本制度旨在通过制度化、常态化的检查评分机制，建立班级常规管理的客观评价体系，实现以下目标：

1. **规范管理**：为班级日常管理提供统一的检查标准和评价依据，使管理工作有章可循；
2. **持续改进**：通过定期检查与数据反馈，帮助各班级及时发现问题、持续优化管理水平；
3. **公平评价**：以客观数据替代主观印象，确保对各班级的评价公平、公正、可追溯；
4. **习惯养成**：以检查促进学生良好行为习惯的培养，营造安全、有序、文明的校园环境。

### 1.2 适用范围

- **年级**：1 年级、2 年级、3 年级
- **班级**：每年级 4 个班，共 12 个班级
- **人员**：全校管理人员、年级负责人、值日教师、班主任

### 1.3 常规管理的作用

班级常规管理是学校教育教学工作的基础保障，其核心作用包括：

- **行为习惯培养**：通过日常检查引导学生养成良好的卫生习惯、学习习惯和文明礼仪；
- **安全保障**：通过课间安全、放学秩序等检查项，及时发现和消除安全隐患；
- **教学秩序维护**：通过课前准备、课堂纪律等检查项，保障正常的教学秩序；
- **家校协同**：通过规范化的数据记录，为家校沟通提供客观依据。

### 1.4 基本原则

| 原则 | 说明 |
|------|------|
| **公平公正** | 所有班级适用统一标准，检查结果如实记录，不因主观因素偏移 |
| **数据驱动** | 以系统化的检查数据为决策依据，减少经验主义和主观判断 |
| **持续改进** | 检查不是目的，改进才是目标；重在发现问题、推动整改 |
| **适度检查** | 每日检查项控制在 3～5 项，避免过度检查影响正常教学秩序 |
| **及时反馈** | 检查结果当日录入系统，确保信息的时效性和可操作性 |

### 1.5 评分方式说明

所有检查评分结果均通过学校班级常规管理评分系统完成录入、汇总和统计分析。评分人员在完成巡查后，需在规定时间内将结果录入系统，系统自动生成各维度的统计报表供各级管理人员查看。

## 第二章 组织架构与角色职责

### 2.1 组织架构概览

```
校级管理者（管理员）
├── 一年级负责人
│   ├── 一年级值日教师
│   ├── 1年级1班 班主任
│   ├── 1年级2班 班主任
│   ├── 1年级3班 班主任
│   └── 1年级4班 班主任
├── 二年级负责人
│   ├── 二年级值日教师
│   ├── 2年级1班 班主任
│   ├── 2年级2班 班主任
│   ├── 2年级3班 班主任
│   └── 2年级4班 班主任
└── 三年级负责人
    ├── 三年级值日教师
    ├── 3年级1班 班主任
    ├── 3年级2班 班主任
    ├── 3年级3班 班主任
    └── 3年级4班 班主任
```

### 2.2 管理员（校级管理者）

#### 职责概述

作为检查制度的最高管理者，负责制定学期检查计划、管理检查指标、统筹全校数据，并对制度执行情况进行监督和调整。

#### 具体任务清单

1. **学期排期**：在学期初制定全学期的检查项排期表，确定常驻检查项和轮换检查项的安排；
2. **周计划管理**：每周审核并确认下周的检查计划，可根据实际需要调整检查项目；
3. **检查项管理**：维护固定检查项的描述和启用状态，根据需要创建动态临增检查项；
4. **数据监控**：每日查看全校检查完成情况、各年级达标率、风险预警等关键指标；
5. **异常处理**：对超期未完成评分、连续预警班级等异常情况进行跟进和协调；
6. **制度优化**：根据数据分析结果，适时调整检查标准和计划安排。

#### 怎样才算做得好

- 学期检查计划覆盖所有检查项，各项目被均衡检查；
- 全校日评完成率达到 100%（每个工作日所有班级均被评分）；
- 周评按时完成率达到 100%；
- 能及时发现和处理预警班级的问题；
- 检查项的常驻/轮换安排合理，根据数据反馈适时调整。

### 2.3 年级负责人

#### 职责概述

作为年级管理的第一责任人，负责本年级的周评工作、年级专属检查项管理，以及指导本年级班主任开展班级管理改进。

#### 具体任务清单

1. **周评执行**：每周五对本年级各班级完成周评（W-1 至 W-5），客观评定综合等级；
2. **动态项管理**：根据年级阶段工作重点，创建和管理本年级的动态临增检查项；
3. **数据分析**：每周查看本年级各班达标率、等级分布、薄弱环节等数据；
4. **班主任指导**：对连续预警或达标率偏低的班级，与班主任沟通并协助制定改进方案；
5. **信息汇报**：就年级整体情况向校级管理者汇报。

#### 怎样才算做得好

- 周评在每周截止时间前全部完成，无漏评；
- 周评等级评定客观准确，有据可查；
- 本年级各班平均达标率稳步提升或保持在较高水平；
- 预警班级能在两周内有明显改善；
- 动态检查项的设置具有针对性，能回应年级实际问题。

### 2.4 值日教师

#### 职责概述

作为日常检查的执行者，值日教师负责按照每日检查计划对所辖年级各班级进行巡查评分，如实记录检查结果。

#### 具体任务清单

1. **日常评分**：每个工作日按当日检查计划，对所负责年级的所有班级逐项评分；
2. **如实记录**：对于不达标项，准确判定严重程度（轻微/一般/严重），必要时填写备注说明具体情况；
3. **及时录入**：所有评分结果在当天完成系统录入；
4. **重点关注**：关注近期频繁不达标的班级和项目，加强巡查力度；
5. **沟通反馈**：发现严重问题时，及时向年级负责人和班主任反馈。

#### 怎样才算做得好

- 评分完成率 100%，无遗漏班级或检查项；
- 评分标准统一，同类问题的判定前后一致；
- 评分风格适中（既不过于宽松也不过于严格），与全校平均水平偏差在合理范围内；
- 不达标项有必要的备注说明，备注率保持在合理水平；
- 当日评分当日录入，不积压到次日。

### 2.5 班主任

#### 职责概述

作为班级管理的直接责任人，班主任需要定期查看本班的检查成绩，根据反馈及时调整管理策略，促进班级达标率提升。

#### 具体任务清单

1. **成绩查看**：每日/每周查看本班的检查成绩、达标率和等级评定；
2. **问题识别**：分析不达标项目的规律（高频不达标项、时间规律等），找出根本原因；
3. **整改落实**：针对不达标项目制定具体整改措施，并在班级内推动执行；
4. **配合检查**：积极配合值日教师的检查工作，不干扰正常检查流程；
5. **持续跟踪**：关注整改效果，确保改进措施的持续性。

#### 怎样才算做得好

- 定期（至少每周一次）查看班级成绩数据；
- 班级日评达标率保持在 80% 以上，周评等级保持在 B 级及以上；
- 不达标项目能在一周内得到有效改善；
- 无连续两周 C 等级（预警）的情况发生；
- 班级成绩呈稳步上升趋势或保持稳定的高水平。

## 第三章 检查制度

### 3.1 日评制度

日评是班级常规管理检查的基础环节，每个工作日由值日教师对各班级进行巡查评分。

#### 3.1.1 检查频率与时间

- **频率**：每个工作日（周一至周五），法定节假日除外
- **检查时段**：上课期间任意时段（具体时段由值日教师根据检查项目特点灵活安排）
- **录入截止**：当日 23:59 前完成系统录入，次日起锁定

#### 3.1.2 检查计划

每日检查不是对所有 9 个日评项目（D-1 至 D-9）全部检查，而是按照检查计划执行：

- **每日检查项数**：3～5 项
- **常驻项**（每日必查，上限 3 项）：学期初确定，整个学期固定检查的项目
- **轮换项**（按周轮转）：按照学期排期表，分组轮换检查
- **动态临增项**：根据阶段重点或突发情况临时添加的检查项目

#### 3.1.3 评分方式

对每个检查项，评定为以下两种结果之一：

| 评定结果 | 含义 |
|---------|------|
| **达标** | 该班级在该检查项上符合规定标准 |
| **不达标** | 该班级在该检查项上未达到规定标准，需进一步区分严重程度 |

不达标时，需进一步选择严重程度（详见第四章）。

#### 3.1.4 快速评分模式

当检查结果为全部达标时，可使用"快速通过"模式一键完成评分。系统会随机抽取一个近期不达标项目作为重点抽查提醒，确保快速评分的质量。

### 3.2 周评制度

周评是对班级一周整体表现的综合评价，每周五由年级负责人或管理员完成。

#### 3.2.1 检查频率与时间

- **频率**：每周一次（周五进行）
- **评分人**：年级负责人（本年级）或管理员（全校）
- **录入截止**：下周一 12:00 前完成录入，逾期锁定（管理员可超期操作）

#### 3.2.2 评分内容

周评包含 5 项指标（W-1 至 W-5），其中：

- **W-1 至 W-4**：事件统计类指标，记录本周发生的特定事件次数
- **W-5**：综合等级评定，综合考虑日评达标率和 W-1 至 W-4 的情况，评定 A/B/C 三个等级

#### 3.2.3 综合等级标准

| 等级 | 名称 | 评定标准 |
|------|------|---------|
| **A** | 卓越 | 本周日评达标率 >= 90%，且无严重或一般不达标，W-1 至 W-4 均为 0 |
| **B** | 良好 | 本周日评达标率 70%～89%，W-1 至 W-4 单项问题 <= 1 起 |
| **C** | 预警 | 本周日评达标率 < 70%，或存在严重不达标，或 W-1 至 W-4 任一项 >= 2 起 |

### 3.3 检查计划制度

#### 3.3.1 学期排期

学期初，管理员制定全学期的检查项排期表：

1. **确定常驻项**：从 D-1 至 D-9 中选择最多 3 项作为常驻项，每日必查
2. **分组轮换项**：其余检查项按组分配到各周，确保学期内各项目被均衡覆盖
3. **每周上限**：每周检查项总数不超过 5 项（常驻项 + 轮换项）

#### 3.3.2 常驻项与轮换项

| 类型 | 说明 | 数量限制 |
|------|------|---------|
| **常驻项** | 每周均需检查的高优先级项目，适用于问题频发或安全相关的检查项 | 最多 3 项 |
| **轮换项** | 按周轮转的检查项目，每组分配连续 2 周检查 | 无固定限制 |

**动态调整规则**：

- 轮换项近 30 天不达标率超过 25% → 建议提升为常驻项
- 常驻项近 30 天不达标率低于 5% → 建议降级为轮换项

#### 3.3.3 动态临增项

当出现以下情况时，管理员或年级负责人可临时添加动态检查项：

- 学校阶段性工作重点调整（如"校服穿着检查周"）
- 突发问题需要专项关注（如"午餐纪律"频繁出现问题）
- 特殊活动期间的额外要求（如"升旗仪式纪律"）

动态项仅在指定日期生效，可设置为全校通用或年级专属。

## 第四章 评分规则与指标体系

### 4.1 日评检查项详解

以下为 9 个固定日评检查项的详细说明：

#### D-1 教室卫生与整理

| 要素 | 内容 |
|------|------|
| **检查标准** | 地面无可见垃圾/污渍，黑板擦净；桌椅偏离不超过 1/3，清洁工具和体育器材定点归位 |
| **达标判定** | 地面整洁、黑板干净、桌椅基本整齐、物品归位 |
| **不达标判定** | 地面有明显垃圾或污渍、黑板未擦、桌椅严重歪斜、工具散乱 |
| **计划类别** | 常驻项（建议每日必查） |
| **常见问题** | 课后未及时打扫、卫生角工具摆放混乱、黑板槽粉笔灰堆积 |

#### D-2 包干区卫生

| 要素 | 内容 |
|------|------|
| **检查标准** | 教室外包干区域（走廊、楼梯等）地面无垃圾，墙面无明显人为污损 |
| **达标判定** | 包干区域地面干净、墙面完好 |
| **不达标判定** | 包干区有垃圾纸屑、墙面有涂鸦或脚印 |
| **计划类别** | 常驻项（建议每日必查） |
| **常见问题** | 走廊角落积灰、楼梯扶手脏污、墙面粘贴物残留 |

#### D-3 课前准备

| 要素 | 内容 |
|------|------|
| **检查标准** | 铃响后 1 分钟内全班就座安静，走动或聊天人数不超过 2 人 |
| **达标判定** | 铃响 1 分钟内绝大部分学生就座安静 |
| **不达标判定** | 铃响 1 分钟后仍有 3 人以上走动或大声聊天 |
| **计划类别** | 轮换项 |
| **常见问题** | 低年级学生纪律意识弱、课间过度兴奋导致无法迅速安静 |

#### D-4 课堂纪律

| 要素 | 内容 |
|------|------|
| **检查标准** | 趴桌、转身聊天、随意插话等违纪行为人数不超过 2 人 |
| **达标判定** | 课堂秩序良好，违纪人数在 2 人以内 |
| **不达标判定** | 违纪人数超过 2 人，课堂秩序混乱 |
| **计划类别** | 轮换项 |
| **常见问题** | 下午课堂精力不集中、趴桌现象多发 |

#### D-5 课间安全

| 要素 | 内容 |
|------|------|
| **检查标准** | 走廊、楼梯、教室内无奔跑追逐、推搡、攀爬栏杆等危险行为（操场正常活动不计） |
| **达标判定** | 室内区域无危险行为 |
| **不达标判定** | 发现学生在走廊奔跑、楼梯推搡或攀爬栏杆等 |
| **计划类别** | 轮换项 |
| **常见问题** | 低年级走廊追逐、楼梯间打闹；注意区分操场正常体育活动 |

#### D-6 眼保健操

| 要素 | 内容 |
|------|------|
| **检查标准** | 音乐响后全班安静，睁眼或讲话人数不超过 2 人 |
| **达标判定** | 全班认真做眼保健操，违规人数 2 人以内 |
| **不达标判定** | 多人睁眼、讲话或不做操 |
| **计划类别** | 轮换项 |
| **常见问题** | 学生敷衍了事、趁老师不在偷看 |

#### D-7 课间操

| 要素 | 内容 |
|------|------|
| **检查标准** | 铃响后 3 分钟内集合完毕，队列安静整齐，动作明显不到位人数不超过全班 1/5 |
| **达标判定** | 集合迅速、队列整齐、动作基本到位 |
| **不达标判定** | 集合超时、队列散乱、大面积动作不到位 |
| **计划类别** | 轮换项 |
| **常见问题** | 集合速度慢、队伍中讲话打闹、做操动作幅度不够 |

#### D-8 文明礼仪

| 要素 | 内容 |
|------|------|
| **检查标准** | 着装整洁，按要求穿校服（校服日）；见师长能主动问好；同学间无骂人、起外号等不文明言行 |
| **达标判定** | 着装规范、礼貌待人、言行文明 |
| **不达标判定** | 未按要求穿校服、对师长不礼貌、存在不文明言行 |
| **计划类别** | 轮换项 |
| **常见问题** | 非校服日着装随意、同学间互起外号、说脏话 |

#### D-9 放学及路队秩序

| 要素 | 内容 |
|------|------|
| **检查标准** | 路队整齐安静，无学生无故逗留；教室已断电、关窗、关门 |
| **达标判定** | 路队有序、教室关闭完好 |
| **不达标判定** | 路队散乱、学生逗留、教室未关闭 |
| **计划类别** | 轮换项 |
| **常见问题** | 放学后学生逗留教室、忘关电灯风扇、路队排列不整齐 |

### 4.2 周评检查项详解

#### W-1 室外课出勤

| 要素 | 内容 |
|------|------|
| **检查内容** | 统计室外课未提前请假且无事后补假的缺勤人次 |
| **评分选项** | 0 人次 / 1 人次 / >=2 人次 |
| **达标标准** | 0 人次为最佳，1 人次可接受 |
| **记录要求** | 如有缺勤需注明具体情况 |

#### W-2 当周安全事故记录

| 要素 | 内容 |
|------|------|
| **检查内容** | 记录需送医务室及以上处理的安全事故起数及处理情况 |
| **评分选项** | 0 起 / 1 起 / >=2 起 |
| **达标标准** | 0 起为目标，1 起需说明处理情况 |
| **记录要求** | 如有事故需注明事故类型和处理结果 |

#### W-3 当周学生冲突记录

| 要素 | 内容 |
|------|------|
| **检查内容** | 记录需教师介入处理或已上报的学生冲突事件 |
| **评分选项** | 0 起 / 1 起 / >=2 起 |
| **达标标准** | 0 起为目标 |
| **记录要求** | 如有冲突需注明事件概况和处理措施 |

#### W-4 当周家长有效反馈/投诉

| 要素 | 内容 |
|------|------|
| **检查内容** | 记录家长通过正式渠道提出的需学校回应或处理的诉求 |
| **评分选项** | 0 起 / 1 起 / >=2 起 |
| **达标标准** | 0 起为目标 |
| **记录要求** | 如有投诉需注明诉求内容和回应情况 |

#### W-5 本周班级整体运行等级

| 要素 | 内容 |
|------|------|
| **检查内容** | 综合本周日评数据和 W-1 至 W-4 情况，评定班级整体运行等级 |
| **评分选项** | A（卓越）/ B（良好）/ C（预警） |
| **必填要求** | 选择等级后必须填写评定说明，阐明评定依据 |

**等级评定详细标准**：

| 等级 | 条件（须同时满足） |
|------|-----------------|
| **A（卓越）** | 本周日评达标率 >= 90%；无严重或一般程度的不达标记录；W-1 至 W-4 均为 0 |
| **B（良好）** | 本周日评达标率 70%～89%；W-1 至 W-4 单项 <= 1 起；无严重不达标 |
| **C（预警）** | 本周日评达标率 < 70%；或存在严重不达标；或 W-1 至 W-4 任一项 >= 2 起 |

### 4.3 不达标严重程度分级

日评检查中，当某班级某检查项被判定为不达标时，需进一步选择严重程度：

| 严重程度 | 说明 | 典型场景 |
|---------|------|---------|
| **轻微** | 问题偶发、影响范围小、可立即纠正 | 地面有少量纸屑、个别学生未按时就座 |
| **一般** | 问题频次较高或影响范围较大、需班主任跟进 | 多处地面脏污、课堂纪律明显松散 |
| **严重** | 涉及安全隐患或严重违规、需即时处理并上报 | 学生攀爬栏杆、教室设备损坏未处理 |

**填写备注的建议**：

- 不达标项建议填写备注，简要描述具体情况（如"3 名学生在走廊奔跑"）；
- 严重不达标必须填写备注，说明具体问题和当场采取的措施；
- 备注内容应客观、具体、简洁，便于后续追踪。

### 4.4 达标率计算

**日评达标率**（日维度）：

$$\text{达标率} = \frac{\text{该日达标的检查项数}}{\text{该日检查的总项数}} \times 100\%$$

**周达标率**（周维度）：

$$\text{周达标率} = \frac{\text{本周所有达标记录数}}{\text{本周所有检查记录数}} \times 100\%$$

### 4.5 动态临增检查项

除固定的 D-1 至 D-9 外，系统支持临时添加动态检查项，常见的动态项包括：

| 动态项示例 | 适用场景 |
|-----------|---------|
| 课桌整理 | 教室整理专项检查 |
| 午餐纪律 | 午餐秩序专项关注 |
| 红领巾佩戴 | 少先队活动期间 |
| 校服穿着 | 校服日专项检查 |
| 垃圾分类 | 环保专题教育期间 |
| 升旗仪式纪律 | 周一升旗专项 |
| 图书角整理 | 阅读活动期间 |
| 节能环保 | 节能宣传周 |

动态项的评分方式与固定项相同（达标/不达标 + 严重程度分级）。

## 第五章 各角色操作指南

### 5.1 管理员操作指南

#### 5.1.1 学期排期

**操作时机**：学期初（开学前一周内）

**操作步骤**：

1. 确定本学期的常驻检查项（建议选择卫生类或安全类等高频检查项，最多 3 项）
2. 将剩余检查项设置为轮换项，系统自动按组分配到各周
3. 审核排期表，确保所有检查项在学期内被均衡覆盖
4. 确认后系统自动生成各周的检查计划

**注意事项**：

- 常驻项变更需谨慎，建议每学期调整不超过 1 次
- 参考系统提供的"调整建议"（基于近 30 天不达标率）来优化常驻/轮换配置

#### 5.1.2 每周计划确认

**操作时机**：每周五或周末

**操作步骤**：

1. 查看系统推荐的下周检查项（基于历史数据和不达标率）
2. 确认推荐方案或手动调整
3. 确认后系统自动生成下周各工作日的检查计划
4. 如需在特定日期增加动态检查项，单独添加

#### 5.1.3 数据监控

**操作频率**：每日查看

**关注重点**：

- 当日检查完成情况（是否所有班级均已评分）
- 全校本周达标率趋势
- 各年级达标率对比
- 预警班级列表
- 值日教师工作状态（是否按时完成评分）

#### 5.1.4 异常处理

| 异常情况 | 处理方式 |
|---------|---------|
| 当日评分未完成 | 联系对应值日教师，督促尽快完成 |
| 连续预警班级 | 约谈年级负责人和班主任，协助制定改进方案 |
| 周评超期 | 管理员可超期完成，但应了解超期原因 |
| 评分异常 | 检查评分数据是否合理，必要时与评分人核实 |

### 5.2 年级负责人操作指南

#### 5.2.1 周评流程

**操作时机**：每周五

**操作步骤**：

1. 进入周评功能，选择待评班级
2. 逐项填写 W-1 至 W-4，如有非 0 项需填写说明
3. 查看系统建议的 W-5 等级（基于本周日评数据自动推荐）
4. 确认等级或根据实际情况手动调整
5. 填写评定说明（必填），阐明等级评定的依据
6. 提交周评

**注意事项**：

- 周评截止时间为下周一 12:00，请务必在截止前完成
- W-5 等级评定应综合考虑日评数据和 W-1 至 W-4 的情况，不应仅凭主观印象
- 评定说明应具体、有据，如"本周达标率 92%，无安全事故和冲突，W-1~W-4 全部为 0，评定为 A"

#### 5.2.2 动态检查项管理

**操作时机**：根据年级工作需要

**操作步骤**：

1. 确定需要临时增加的检查项目和生效日期
2. 创建动态检查项，填写项目名称和检查标准
3. 设定年级范围（本年级专属或全校通用）
4. 创建后该项会自动出现在当日检查计划中

#### 5.2.3 年级数据分析

**操作频率**：每周至少查看一次

**关注重点**：

- 本年级各班达标率排名
- 薄弱检查项（不达标率高于 15% 的项目）
- 各班等级变化趋势
- 与上周的对比数据

### 5.3 值日教师操作指南

#### 5.3.1 日常评分流程

**操作时机**：每个工作日

**操作步骤**：

1. 查看当日检查计划，了解今天需要检查哪些项目
2. 按检查项目逐个班级进行实地巡查
3. 进入评分页面，选择班级
4. 对每个检查项评定为"达标"或"不达标"
5. 不达标项选择严重程度，建议填写备注
6. 确认无误后提交评分
7. 依次完成所有班级的评分

**巡查建议**：

- 根据检查项特点选择合适的巡查时段（如"课前准备"应在上课铃响后检查）
- 建议从高楼层向低楼层巡查，提高效率
- 对于近期频繁不达标的班级，可适当增加巡查次数

#### 5.3.2 快速评分

**适用场景**：经巡查确认该班级当日所有检查项均达标

**操作步骤**：

1. 选择班级后点击"快速通过"
2. 系统会随机抽取一个近期不达标的项目作为重点提醒
3. 确认该抽查项确实达标后提交

**注意事项**：

- 快速评分不是"免检"，必须在实际巡查确认全部达标后才能使用
- 如有任何一项不达标，应改用逐项评分模式

#### 5.3.3 评分画像解读

系统会生成个人"检查画像"，反映评分风格和工作量：

| 指标 | 说明 | 参考范围 |
|------|------|---------|
| **个人达标率** | 您评分的整体达标比例 | 与全校/年级平均对比 |
| **评分风格** | 综合判定评分严格程度 | 偏松/适中/偏严 |
| **严重程度分布** | 不达标记录中轻微/一般/严重的比例 | 无固定标准，供自我参照 |
| **备注率** | 填写备注的记录占比 | 建议不达标项备注率 > 50% |
| **检查天数** | 本学期已完成评分的工作日数 | 与应检查天数对比 |

### 5.4 班主任操作指南

#### 5.4.1 成绩查看

**操作频率**：建议每日查看，至少每周一次

**可查看内容**：

- 本班当日/本周/本月/本学年的检查记录
- 各检查项的达标率（柱状图展示）
- 周评记录和等级历史
- 达标率变化趋势

#### 5.4.2 问题识别

**分析维度**：

1. **高频不达标项**：哪些检查项目经常不达标？是某个固定项还是多项？
2. **时间规律**：是否存在特定时间段（如周一、下午）不达标率偏高的情况？
3. **严重程度**：不达标记录中"严重"占比是否偏高？
4. **趋势变化**：达标率是在改善还是恶化？

#### 5.4.3 改进建议

| 常见问题 | 建议措施 |
|---------|---------|
| 教室卫生频繁不达标 | 建立值日轮换制度，明确责任区域，放学前检查 |
| 课间安全频繁不达标 | 加强安全教育，安排课间文明监督员 |
| 课前准备不达标 | 培养课间整理习惯，预备铃提前准备 |
| 文明礼仪不达标 | 开展主题班会教育，设立文明之星评选 |
| 课间操不达标 | 利用班会课练习，体育课加强队列训练 |

## 第六章 激励与预警机制

### 6.1 流动红旗

**评定标准**：连续 2 周及以上获得 A（卓越）等级的班级

**意义**：

- 是对班级持续优秀表现的认可
- 在全校通报表扬，树立先进典型
- 激发其他班级的赶超意识

### 6.2 连续预警

**触发标准**：连续 2 周及以上获得 C（预警）等级的班级

**处理措施**：

1. 年级负责人与班主任面谈，分析原因
2. 制定限期整改方案
3. 加大巡查频次，重点关注
4. 跟踪整改效果，直至脱离预警状态

### 6.3 显著进步

**评定标准**：较上周等级提升的班级（如 C 升至 B，或 B 升至 A）

**意义**：

- 肯定班级的改进努力
- 鼓励持续保持良好态势
- 在全校通报中予以正面反馈

### 6.4 等级变化趋势

系统自动追踪各班级的等级变化趋势：

| 趋势标记 | 判定依据 |
|---------|---------|
| **上升** | 本周达标率高于上周 |
| **下降** | 本周达标率低于上周 |
| **稳定** | 达标率变化幅度在 5% 以内 |
